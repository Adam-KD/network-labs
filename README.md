# Network Labs

A collection of hands-on networking labs built in Cisco Packet Tracer, documenting practical implementation of core networking concepts with a security-conscious approach - VLAN segmentation, trunk hardening, routing, and access control - as part of CCNA certification prep and broader network security skill development.

## About

Each lab is built progressively, starting from foundational Layer 2 concepts and expanding toward full enterprise-style network design including routing, services, and security hardening. Every lab includes the topology, exported device configurations, and a written breakdown of what was implemented and learned.

Built as part of ongoing CCNA (200-301) certification preparation, alongside a Security+ certification and SOC/threat intelligence analyst career track. Networking fundamentals directly support log analysis, traffic inspection, and infrastructure security work.
